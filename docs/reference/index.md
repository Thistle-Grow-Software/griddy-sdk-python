# API Reference

Auto-generated API documentation for the Griddy SDK.

The SDK is organized into the following domains:

- **[NFL](nfl/)** — NFL.com API client (Regular, Pro, and Next Gen Stats endpoints)
- **[PFR](pfr/)** — Pro Football Reference HTML scraping client
- **[DraftBuzz](draftbuzz/)** — Draft prospect data scraping client
- **[Core](core/)** — Shared infrastructure (HTTP client, exceptions, utilities)
